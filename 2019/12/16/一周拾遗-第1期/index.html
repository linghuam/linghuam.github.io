如何获取浏览器帧率

无线性能优化：FPS 测试

Web 动画帧率（FPS）计算

帧率即屏幕的刷新频率，大部分设备为 60次/秒，帧率越大动画越流畅。

浏览器中一个 Tab 页对应一个进程，进程 = 主线程(负责计算) + 合成线程(负责绘制)。

js 动画需要主线程和合成线程共同工作，css 动画只需要合成线程工作。
