git-将本地仓库上传到github

在开发中，我们通过git init命令在本地创建了一个版本库，但是如何把它上传到 github 上共享呢。

其实这个问题再延伸一下，就是如何关联远程仓库和本地仓库。分以下几种情况：

  • github有，本地无
  • github无，本地有
